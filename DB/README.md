2. 컬럼에 있는 내용 중 숫자만 뽑기

3. Group by로 중복 찾기
```
SELECT COLUMN_NAME , COUNT(*) FROM TABLE_NAME GROUP BY COLUMN_NAME HAVING COUNT(*) > 1 ;
```
